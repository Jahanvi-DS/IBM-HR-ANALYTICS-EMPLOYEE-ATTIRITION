# IBM HR Analytics - Employee Attrition Case Study

📌 Project Overview
This project focuses on exploring and analyzing the factors that lead to employee attrition in an organization using the standard IBM HR Analytics dataset. By understanding these patterns, organizations can take proactive measures to retain high-performing talent.

📊 Key Statistical and EDA Insights Discovered
- Structure and Quality Check: Checked dataset integrity using df.info(), df.describe(), and verified that there are zero missing values using df.isnull().sum().
- Income Impact: Employees with lower monthly income show significantly higher attrition rates, indicating compensation is a key retention driver.
- Age Trend: Younger workforce in their early career phase is more prone to leaving the company compared to senior employees.

⚙️ Technologies Used
- Language: Python
- Libraries: Pandas, NumPy, Matplotlib, Seaborn
- Environment: VS Code (Jupyter Notebook)

📂 Repository Structure
- IBM_Employee_Attrition_Analysis.ipynb: Complete Jupyter Notebook containing data quality checks, statistical summary, and visualizations.
- WA_Fn-UseC_-HR-Employee-Attrition.csv: The benchmark dataset used for this analysis.
